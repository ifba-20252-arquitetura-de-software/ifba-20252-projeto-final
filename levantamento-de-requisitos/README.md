# Levantamento de Requisitos
### **Assistente Virtual**
* Bot para responder perguntas aos usuários sobre o IFBA e seus cursos.
* Todas as informações estão em repositório.
* O assistente irá trazer um menu de opções e o usuário irá clicar até
encontrar a solução.
* Questões abertas podem existir porém como última opção.
* A interface deve seguir a estilização de site do IFBA.
* Deve haver uma interface admin para que possam ser cadastradas
novas perguntas e respostas.
